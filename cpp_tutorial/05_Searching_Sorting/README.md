# Module 05: Searching & Sorting

## Overview
Searching and Sorting are the bread and butter of algorithms.

## Key Concepts

### 1. Searching
- **Linear Search**: Check every element. Slow (O(n)).
- **Binary Search**: Divide and conquer. Fast (O(log n)), but requires a *sorted* array.

### 2. Sorting
- **Bubble Sort**: Simple but slow (swapping neighbors).
- **Selection Sort**: Find minimum, swap.
- **Merge Sort / Quick Sort**: Much faster O(n log n) algorithms using recursion.
