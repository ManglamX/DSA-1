# Module 10: Graphs

## Overview
Graphs model connections.

## Key Concepts

### 1. Representation
In Java, we usually use an **Adjacency List**.
```java
// List of Lists
List<List<Integer>> adj = new ArrayList<>();
```

### 2. Traversals
- **BFS**: Uses `Queue<Integer>`.
- **DFS**: Uses Recursion or `Stack<Integer>`.
