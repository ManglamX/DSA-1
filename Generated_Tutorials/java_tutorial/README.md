# Java Data Structures & Algorithms Lab

Welcome to your personalized Java DSA Lab! This repository mirrors the C++ lab but focuses on Java's specific strengths, like its Collections Framework and Object-Oriented design.

## Structure
Each module contains:
- `README.md`: Concept explanation.
- `src/`: Tutorial code.
- `exercises/`: Problems to solve.
- `solutions/`: Reference answers.

## Prerequisites
- Java Development Kit (JDK) installed (version 8+ recommended).
- To compile and run a file:
  ```bash
  javac FileName.java
  java FileName
  ```
