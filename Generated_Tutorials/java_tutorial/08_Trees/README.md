# Module 08: Trees & BST

## Overview
Trees are recursive structures consisting of nodes with children. 

## Key Concepts

### 1. Binary Tree
Every node has at most 2 children (`left`, `right`).

### 2. Binary Search Tree (BST)
A specific type of Binary Tree where:
- Left Subtree values < Node value
- Right Subtree values > Node value
